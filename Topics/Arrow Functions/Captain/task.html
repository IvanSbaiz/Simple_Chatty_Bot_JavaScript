Captain

Define a function that will add the word "captain" before the name of a person. There should be a space after the word "captain".

The function should be named captainAdder and accept one parameter name as an argument.

The function should return a string, not print it to the console.


Sample Input:
Jack Sparrow

Sample Output:
captain Jack Sparrow



Memory limit: 256 MB
Time limit: 15 seconds
